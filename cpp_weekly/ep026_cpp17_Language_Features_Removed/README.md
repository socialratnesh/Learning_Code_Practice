=== A few Language Features Removed in C++17 ===

auto_ptr, 
deprecated function objects, 
std::random_shuffle, 			// alternate: shuffle
std::unexpected, 
the obsolete iostreams aliases, 
trigraphs, 
the register keyword, 
bool increment
